# salsa-amelia-portfolio
Portfolio website and repository for **Salsa Amelia Septiana** — Telecommunication Engineering student.

## What is included
- `index.html` — main portfolio landing page
- `styles.css` — stylesheet used by index.html
- `/projects` — suggested folder for detailed project artifacts and READMEs
- `/assets` — placeholder for images and certificates
- `README.md` — this file
- `LICENSE` — MIT license

## How to use
1. Clone or download this repository.
2. Edit `index.html` to update text, links, or contact info.
3. Add project artifacts under `/projects/<project-name>/` and update links in the Documentation section.
4. Commit and push to GitHub. To publish with GitHub Pages:
   - Create a repository named `username.github.io` (replace `username`), or enable GitHub Pages from repository settings.
   - Push the contents to the repository's `main` branch.
   - The site will be available at `https://username.github.io`.

## Deploy alternatives
- **Netlify**: drag & drop the repository zip or connect to GitHub.
- **Vercel**: connect to GitHub and deploy.
